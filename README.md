# Teemo
[![Build Status](https://travis-ci.org/GallenHu/Teemo.svg?branch=master)](https://travis-ci.org/GallenHu/Teemo)

A simple navigation site.

### dev
```sh
yarn # npm install
npm run dev
```
open `http://127.0.0.1:8000/`

### build / publish
```sh
npm run build
```
